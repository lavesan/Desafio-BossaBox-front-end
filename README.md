## Prerequisites
1. node@8+
2. create-react-app@2.1.8
3. git clone and run this repo:
    * https://gitlab.com/bossabox/challenge-fake-api/tree/master

## Available Scripts
To Run the project in development mode:
### `npm run start`
To Test the project:
### `npm run test`
